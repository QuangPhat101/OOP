# Vẽ sơ đồ UML
## Mũi tên
- Mũi tên tam giác A -> B: B là cha của A (A is-a B)
- Mũi tên hình thoi A -> B: B sỡ hữu A (B has-a A) ở đầu A có thể có số hoặc dấu * chỉ số lượng
## Vẽ các class
- Kí hiệu
+ "#": Protected
+ "-": Private
+ "+": Public
- Thuộc tính:
VD: - _name : string
- Hành vi:
VD1: salary() : int
VD2: <virtual> salary(): int -> <override> salary(): int
